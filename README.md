![landing_page]

## 📖 About the Project

This is a powerful web application designed to detect human faces and accurately predict both their gender and approximate age range. It combines a user-friendly web interface with robust backend processing.

The core technology stack includes **Flask**, **OpenCV**, **HTML**, and **CSS**.

---

## 🚀 Key Features

* 📺 **Real-time Detection:** Process live video feeds from your webcam.
* 📸 **Image Analysis:** Upload static images (.jpg, .png) for quick detection.
* 🧠 **Deep Learning Models:** Utilizes pre-trained deep learning models for high-accuracy gender classification and age estimation.
* 💻 **Responsive Design:** A clean and modern user interface that works seamlessly on desktop browsers.

## ⚙️ Getting Started

Follow these steps to set up and run the project locally.

### Prerequisites

Ensure you have Python 3.x installed. The project depends on the following libraries:

* OpenCV (`opencv-python`)
* Flask (`Flask`)
* Pillow (`Pillow`)

### Installation

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/your-username/Gender-Age-Predictor.git](https://github.com/your-username/Gender-Age-Predictor.git)
    cd Gender-Age-Predictor
    ```

2.  **Install Prerequisites:**
    You can install them individually:
    ```bash
    pip install Flask opencv-python Pillow
    ```
    *(Optional but recommended: create a `requirements.txt` file)*.

### Execution Guide

1.  **Start the Server:**
    Open your terminal in the project directory and run:
    ```bash
    python app.py
    ```

2.  **Access the Application:**
    Open your web browser and navigate to:
    ```
    [http://127.0.0.1:5000/](http://127.0.0.1:5000/)
    ```
